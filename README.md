## Summary
- AnalystFolio is a fully responsive personal portfolio website developed to showcase my journey, projects, certifications, and technical skills as a Data Analyst & Data Science enthusiast.  
- This project reflects my hands-on learning in web development, problem-solving, GitHub deployment, and portfolio presentation while building a professional online presence from scratch.


## 🛠️ Technologies  
- HTML5  
- CSS3  
- JavaScript  
- Bootstrap  
- Font Awesome  
- Git & GitHub  
- GitHub Pages


## ✨ Features
- Responsive portfolio design  
- Smooth scrolling navigation  
- Animated sections and transitions  
- Skills section with technology logos  
- Project showcase with GitHub links  
- Certification section with certificate links  
- Social media integration  
- Resume download button  
- Clean modern UI design  
- Hosted live using GitHub Pages


## ⚙️ The Process
- Planned the portfolio structure and sections  
- Customized an existing template according to my profile  
- Added personal branding and project information  
- Integrated project repositories and certifications  
- Fixed image path and deployment issues on GitHub  
- Optimized responsiveness and UI spacing  
- Added social media icons and external links  
- Deployed the website successfully using GitHub Pages


## 📚 What I Learned
- Hosting websites using GitHub Pages  
- Managing repositories and deployment structure  
- Fixing image path and file linking issues

- Improving website responsiveness and layout handling  
- Using Font Awesome icons and animations  
- Real-world debugging and troubleshooting  
- Building a professional online portfolio independently
